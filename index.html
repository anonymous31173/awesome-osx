Productivity

  • Homebrew (Free) (GitHub) - The missing package manager ...
  • 1Password ($99) - Password / Credit Card manager
    • SudoLikeABoss (Free) - Plugin to iTerm, fill passwords on your terminal with a keyboard shortcut.
  • f.lux (Free) - Controlling screen 'temperature'
  • Postman (Freemium) - Web Request manager, to test your web-services, API, ...
  • Dash (Freemium) - Instant offline access to API documentation sets and code snippet manager on Mac. Really awesome tools for development.
  • Alfred (Freemium) - Shortcuts manager.

Windows arrangers

  • Spectacle (Free) (GitHub) - Keyboard window resizer / arranger.
  • BetterSnapTool (Free) - Another keyboard windows resizer / arranger.

Status tools

  • Memory Clean (Freemium) - RAM Manual handler, get full control of your RAM.
  • AppCleaner (Freemium) - Clean apps and their files.

Terminal

  • iTerm (Free) - Powerful terminal emulator.
    • SudoLikeABoss (Free) - Plugin to iTerm, fill passwords on your terminal with a keyboard shortcut.
  • Cathode ($7.99) - Another awesome terminal emulator

Shell

  • zsh (Free) - Powerful shell that operates as both an interactive shell and as a scripting language interpreter
    • oh-my-zsh (Free) - Oh-My-Zsh is an open source, community-driven framework for managing your ZSH configuration. It comes bundled with a ton of helpful functions, helpers, plugins and themes.
  • fish (Free) - Smart and user-friendly command line shell for OS X, Linux, and the rest of the family.

Editors

  • Sublime Text (Freemium) - Cool code editor.
  • Text Mate (Free)- OSX Missint code editor.
  • Atom (Free) (GitHub) - 21s code editor.

IDEs

  • NetBeans (Free) - Powerful IDE for Java, HTML5, PHP, C/C++ ...
  • Eclipse (Free) - Powerful IDE for Java, PHP ,C++ ...
  • IntelliJ (Freemium) - Smart and powerful Java IDE.
  • Android Studio (Free) Powerful IDE for Android (IntelliJ Based).
  • WebStorm ($99) - Powerful IDE for NodeJS.
  • RubyMine ($99) - Powerful IDE for Ruby.

Database

  • Postgres App (Free) - The easiest way to get started with PostgreSQL on the Mac, this way you can start/stop Postgres at the Mac OS X menubar. It comes with batteries included.
  • Sequel Pro (Free) - MySQL management GUI specific for Mac OS X.
  • Sqlite Browser (Free) - Sqlite management GUI with binaries included for Mac OS X.
  • RoboMongo (Free) - MongoDB management GUI mixed with the full power of the shell commands. Includes binaries for Mac OS X.

Virtualization

  • Kitematic (Free) - The easiest way to use Docker on Mac.

Design

  • Prepo (Freemium) - No more pain creating all necessary sizes of icons when developing a App for Android an iOS.
  • Sketch ($99) - The best software to design user interfaces and with a great community developing plugins and tutorials.
  • Pixelmator for Mac ($29.99) - Full-featured image editing app for the Mac and much more cheap than Adobe solution.

Teams

Communication

  • Sococo - Remote team communication software, support video/audio and screen-sharing
  • Slack - Team communication software with several integrations for development tools, such as github, gitlab, jenkins, ...
